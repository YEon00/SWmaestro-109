# 5. Main Function

* 낙상감지를 통하여 응급상황 시 관제 센터 및 보호자 알림
* 능동적 말벗 기능을 통하여 외로움 완화 및 친근한 대화 유도
* 온/습도 등 집 상태를 모니터링 하여 응급상황 시 센터에 자동 신고 기능
* 약 복용 알림 기능
* 보호자 앱
* 수집한 데이터를 시각화하는 대시보드 구현
